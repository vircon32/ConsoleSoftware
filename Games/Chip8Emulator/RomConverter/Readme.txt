=================================================
  CHIP-8 TO VIRCON32
=================================================


What is this?

    This is a program that allows you to convert Chip-8 and
    Super Chip-8 ROM files into Vircon32 memory card files, to
    make them loadable with the Chip-8 emulator for Vircon32.


How to use it?

    Just run this program from the command line, with the
    following arguments:
    
        chip8tovircon32 <input file> <output file>
    
    being the input file a Chip-8 or Super Chip-8 rom file.
    The output will be a Vircon32 memory card file that you
    can load into a Vircon32 emulator as rom for your Chip-8
    emulator.
