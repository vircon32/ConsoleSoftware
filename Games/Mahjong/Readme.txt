  NOTES:
---------------------------
    This version of Mahjong solitaire includes some quality
    of life features to make your games more enjoyable:
    
    - All games are guaranteed to be winnable.
    - You can undo and redo a certain number of matches.
    - The hint feature will cycle through all available pairs.
